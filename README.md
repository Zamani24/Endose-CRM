# Endose CRM

Custom design CRM for Endose Architectural Design.

## Setup

1. Install dependencies: `npm install`
2. Run locally: `npm run dev`
3. Build for production: `npm run build`

## Deploy to Netlify

1. Go to netlify.com → Add new site → Import from Git
2. Connect your GitHub repo
3. Build command: `npm run build`
4. Publish directory: `dist`
5. Click Deploy

## First Login

After deploying, go to your Supabase dashboard:
1. Authentication → Users → Invite user
2. Send invite to rabia@endose.ca
3. She sets her own password via email link

## Supabase Setup

Run the schema SQL in Supabase SQL Editor before first use.
